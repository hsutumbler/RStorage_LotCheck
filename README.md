# 試劑入庫管理系統

一個專業的試劑庫存管理解決方案，具備現代化的Web介面和完整的入庫管理功能。

## 功能特色

### 🏷️ 入庫管理
- **新增入庫資料**：記錄試劑名稱、新批號、舊批號、數量等資訊
- **自動標籤列印**：儲存時自動生成並列印標籤，列印數量等於入庫數量
- **批號類型識別**：自動識別新批號與舊批號，並給予相應提醒

### 🔍 搜尋與查詢
- **多條件搜尋**：支援試劑名稱、批號等關鍵字搜尋
- **入庫記錄查看**：完整的入庫歷史記錄，包含所有必要資訊
- **即時更新**：資料即時同步，確保資訊準確性

### 🖨️ 標籤管理
- **自動列印**：入庫時自動列印對應數量的標籤
- **補印功能**：支援重新列印標籤，可自訂列印數量
- **標籤內容**：包含試劑名稱、批號、數量、日期等完整資訊

### 📱 使用者介面
- **現代化設計**：採用Bootstrap 5框架，介面美觀且響應式
- **繁體中文**：完整的中文介面，符合台灣使用者習慣
- **直觀操作**：簡潔明瞭的操作流程，提升使用體驗

## 系統需求

- Python 3.7+
- 現代化瀏覽器（Chrome、Firefox、Safari、Edge）

## 安裝與設定

### 1. 安裝Python套件
```bash
pip install -r requirements.txt
```

### 2. 執行程式
```bash
python app.py
```

### 3. 開啟瀏覽器
在瀏覽器中開啟 `http://localhost:5000`

## 使用說明

### 新增入庫資料
1. 在「新增入庫資料」區塊填寫必要資訊
2. 必填欄位：試劑名稱、新批號、入庫數量
3. 選填欄位：舊批號、備註
4. 點擊「儲存並列印標籤」完成入庫

### 搜尋入庫資料
1. 在搜尋框輸入關鍵字（試劑名稱、批號等）
2. 按Enter鍵或點擊搜尋按鈕
3. 系統會顯示符合條件的記錄

### 補印標籤
1. 在入庫記錄表格中找到需要補印的項目
2. 點擊「補印」按鈕
3. 設定要列印的數量
4. 確認列印

## 資料庫結構

系統使用SQLite資料庫，包含以下欄位：
- `id`：唯一識別碼
- `reagent_name`：試劑名稱
- `old_batch_number`：舊批號
- `new_batch_number`：新批號
- `quantity`：入庫數量
- `entry_date`：入庫日期
- `entry_type`：批號類型（新/舊）
- `notes`：備註

## 技術架構

- **後端**：Flask (Python)
- **資料庫**：SQLite + SQLAlchemy
- **前端**：HTML5 + CSS3 + JavaScript
- **UI框架**：Bootstrap 5
- **圖示**：Font Awesome
- **PDF生成**：ReportLab

## 注意事項

1. **新批號提醒**：系統會自動識別新批號並給予提醒，請確認資訊正確性
2. **標籤列印**：目前系統會生成PDF檔案，實際列印需要整合印表機驅動程式
3. **資料備份**：建議定期備份SQLite資料庫檔案
4. **瀏覽器相容性**：建議使用最新版本的現代化瀏覽器
