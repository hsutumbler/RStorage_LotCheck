# 試劑入庫管理系統

一個專業的試劑庫存管理解決方案，具備現代化的Web介面和完整的入庫管理功能。

## 功能特色

### 🏷️ 入庫管理
- **新增入庫資料**：記錄試劑名稱、批號、穩定效期、數量、單位、供應商等資訊
- **自動標籤列印**：儲存時自動列印5cm x 3.5cm標籤，直接發送到預設印表機
- **新批號提醒**：自動識別新批號並給予確認提醒
- **CSV匯入功能**：支援批量匯入現有試劑資料

### 🔍 搜尋與查詢
- **整合式介面**：搜尋與紀錄功能整合在同一頁面
- **多條件篩選**：支援試劑名稱、供應商、日期範圍等篩選
- **即時更新**：資料即時同步，確保資訊準確性
- **資料匯出**：支援CSV格式匯出功能

### 🖨️ 標籤管理
- **直接列印**：無需PDF檢視器，直接列印到Windows預設印表機
- **標準規格**：固定5cm x 3.5cm標籤尺寸，適合實驗室使用
- **完整資訊**：包含試劑名稱、批號、穩定效期、入庫時間、出庫欄位
- **補印功能**：支援重新列印標籤，可自訂列印數量

### 📱 使用者介面
- **現代化設計**：採用Bootstrap 5框架，介面美觀且響應式
- **繁體中文**：完整的中文介面，符合台灣使用者習慣
- **無縫操作**：靜默處理，無多餘對話視窗打擾

## 系統需求

- Python 3.7+
- 現代化瀏覽器（Chrome、Firefox、Safari、Edge）

## 安裝與設定

### 1. 安裝Python套件
```bash
pip install -r requirements.txt
```

### 2. 執行程式
**Windows用戶：**
雙擊 `啟動程式.bat` 即可自動啟動系統

**手動啟動：**
```bash
python simple_app.py
```

### 3. 開啟瀏覽器
系統會自動開啟瀏覽器到 `http://localhost:5000`

## 使用說明

### 新增入庫資料
1. 在「新增入庫」頁籤填寫必要資訊
2. 必填欄位：試劑名稱、批號、穩定效期、數量、單位、供應商
3. 點擊「儲存並自動列印標籤」完成入庫
4. 系統會靜默處理，無對話視窗

### 搜尋與查詢
1. 切換到「搜尋與紀錄」頁籤
2. 使用篩選器設定搜尋條件
3. 支援試劑名稱、供應商、日期範圍篩選
4. 點擊「套用篩選」查看結果

### 補印標籤
1. 在「搜尋與紀錄」頁面找到需要補印的項目
2. 點擊該項目的「列印」按鈕
3. 系統會直接列印標籤到預設印表機

### CSV匯入功能
1. 在「搜尋與紀錄」頁面點擊「匯入」按鈕
2. 下載CSV範本並填入資料
3. 選擇填好的CSV檔案匯入
4. 系統會自動更新或新增記錄

## 資料庫結構

系統使用SQLite資料庫，包含以下欄位：
- `id`：唯一識別碼
- `reagent_name`：試劑名稱
- `reagent_batch_number`：試劑批號
- `expiry_date`：穩定效期
- `quantity`：入庫數量
- `unit`：單位
- `supplier`：供應商
- `entry_date`：入庫日期

## 技術架構

- **後端**：Flask (Python)
- **資料庫**：SQLite + SQLAlchemy
- **前端**：HTML5 + CSS3 + JavaScript
- **UI框架**：Bootstrap 5
- **圖示**：Font Awesome
- **PDF生成**：ReportLab

## 注意事項

1. **新批號提醒**：系統會自動識別新批號並給予確認提醒
