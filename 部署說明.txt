試劑入庫管理系統 - 部署說明
================================

📁 部署方式
-----------
將整個 RStorage_LotCheck 資料夾複製到目標機台即可

🖥️ 系統需求
-----------
• 作業系統：Windows 10/11
• Python：3.8 或更新版本
• 記憶體：至少 4GB RAM
• 硬碟空間：至少 100MB 可用空間

📦 安裝步驟
-----------
1. 複製資料夾
   - 將 RStorage_LotCheck 資料夾複製到目標機台
   - 建議放在 C:\ 或 D:\ 根目錄

2. 安裝 Python（如果尚未安裝）
   - 下載 Python 3.8+ 從 https://www.python.org/
   - 安裝時勾選 "Add Python to PATH"

3. 安裝必要套件
   - 開啟命令提示字元（CMD）
   - 切換到 RStorage_LotCheck 資料夾
   - 執行：pip install -r requirements.txt

4. 啟動系統
   - 雙擊 quick_start.bat
   - 等待 3-5 秒系統啟動
   - 瀏覽器會自動開啟到 http://127.0.0.1:5000

🚀 快速啟動
-----------
• 日常使用：雙擊 quick_start.bat
• 停止服務：雙擊 stop_service.bat
• 完整安裝：雙擊 start.bat（首次使用）

🔧 故障排除
-----------
問題 1：Python 未安裝
解決：下載並安裝 Python 3.8+

問題 2：套件安裝失敗
解決：執行 pip install --upgrade pip 後重試

問題 3：埠號 5000 被占用
解決：雙擊 stop_service.bat 後重新啟動

問題 4：瀏覽器無法開啟
解決：手動開啟瀏覽器，輸入 http://127.0.0.1:5000

問題 5：中文字體顯示異常
解決：確認系統已安裝微軟正黑體

📋 功能說明
-----------
• 新增入庫：輸入試劑資料並自動列印標籤
• 搜尋紀錄：按名稱、供應商、日期篩選
• 補印標籤：重新列印已入庫的標籤
• 匯出資料：下載 CSV 格式的入庫紀錄

🖨️ 列印設定
-----------
• 系統會自動使用 Windows 預設印表機
• 支援實體印表機、PDF 印表機、網路印表機
• 標籤尺寸：5cm x 3.5cm
• 中文字體：微軟正黑體

📞 技術支援
-----------
如遇問題，請檢查：
1. Python 版本是否正確
2. 必要套件是否安裝完成
3. 防火牆是否阻擋應用程式
4. 埠號 5000 是否被其他程式占用

✅ 部署完成檢查
--------------
1. 雙擊 quick_start.bat 後無錯誤訊息
2. 瀏覽器能正常開啟 http://127.0.0.1:5000
3. 能正常新增入庫資料
4. 標籤能正常列印
5. 搜尋功能正常運作
